1= What is DOM and what is the purpose of DOM?

Ans : DOM stands is Document Object Model.The purpose of the DOM is to provide a standardized way for programming languages to interact with HTML

2= How will you select HTML elements using DOM? Name the DOM methods.

ans : There are several ways to select HTML elements using the DOM. The following are some of the most commonly used DOM methods for selecting HTML elements:

1)getElementById()

2)getElementsByClassName()

3)getElementsByTagName()

4)querySelector()

5)querySelectorAll()

3= What is event bubble?

ans : Event bubbling is a method of event propagation in the html dom api. It is a process that starts with the element that triggered the event and then bubbles up to the elements in the hierarchy.
